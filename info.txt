    Введение
        Данный проект разрабатывается в рамках предмета "Технологии разработки программного обеспечения".
        Он предназначен в первую очередь для отработки командной работы по разработке готового продукта
        Проект не рассчитан на массовую аудиторию, хотя его наработки возможно перерастут в что-то более завершенное.

    Общее описание
        Данный проект представляет собой простое приложения для Android для быстрого обмена сообщениями с людьми вокруг.
        Здесь проскакивает аналогия с простым криком - приложение практически так же работает.
        Основной функционал это возможность отослать в сеть простое сообщение и люди с этим же приложением в определенном радиусе получат уведомление
        об этом и смогут посмотреть на карте откуда это прокричали.

    Функциональность системы
        Радиус в котором пользователь "слушает" других пользователей может настраиваться.
        Так же возможно добавить фильтры по сообщениям. Возможно в дальнейшем раздель их на каналы по типу чатов.
        Приложение будет получать/отправлять всю информацию для своего функционирования через API выделенного сервера в сети.
        
    Используемые языки и фреймворки
        Писать будем под Android 5.0 Lollipop API 21 (Покрытие 85% пользователей по оценке Google)
        Фреймворки думаю  по мелочи Retrofit и RxJava, а в основном натив.
        Для тестирования JUnit.

    Нефункциональные требования
        Данные и сообщения должны передаваться только по зашифрованному каналу.
        Возможно сохранение истории на Google Drive владельца в зашифрованном виде.
